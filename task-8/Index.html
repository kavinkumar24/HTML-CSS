Types of Combinators

Selects all descendants (children, grandchildren, etc.) of a specified element.

Example: div p selects all p tags inside a div tag.

Selects only direct children of a specified element.

Example: div > p selects p tags that are direct children of a div tag.

Selects an element immediately following a specified element.

Example: h1 + p selects the first p tag immediately after any h1 tag.

Selects all siblings of a specified element, even if they are not immediately following it.

Example: h1 ~ p selects all p elements that are siblings of any h1 tag.
